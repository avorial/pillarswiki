# Character JSON Exports

Character sheets are stored under their Drive clan folder path, for example:

```text
characters/Tremere/<character-sheet>.json
characters/Brujah/<character-sheet>.json
```

Each JSON file keeps:

- normalized character fields from the print sheet
- player fields, including Gmail/account/coordinator fields when present
- attributes, backgrounds, pools, abilities, status, disciplines, merits/flaws, rituals, and specializations
- raw parsed data for every workbook tab, including histories, XP spends, personal plots, blood bonds, downtimes, submissions, and other unique pages
